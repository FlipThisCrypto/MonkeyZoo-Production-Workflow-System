# Emo Monkeys: The Signal Between Us

This folder is the filed home for the August 2026-January 2027 Emo Monkeys season bible.

- `SEASON-BIBLE.md` is the narrative season plan.
- `season-metadata.yaml` is the machine-readable season header.
- `CHARACTER-AND-REFERENCE-INDEX.md` lists who each character is and where production should find references.
- `continuity-map.md`, `echo-mystery-tracker.md`, `character-growth-tracker.md`, and `guest-appearance-tracker.md` give production quick-reference views.
- `issue-01-neonblue.md` through `issue-06-twotone.md` are the per-issue working stubs.
- Character canon still lives in `character-bibles/MZ-CHAR-*/bible.yaml`.
- Official website context is summarized in `source_of_truth/official-monkeyzoo-web-sources.md`.
- Production canon and hard rules still live in `00_SYSTEM/`.
