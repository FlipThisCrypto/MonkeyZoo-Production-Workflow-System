# Character Differentiation Guide

Use this guide before adding a trait from the library.

## Differentiation Checks

Compare the candidate trait against the current cast:

- Personality: Does another character already carry this emotional function?
- Speech: Would the line sound like another character?
- Story role: Does this give two characters the same job in the plot?
- Visual identity: Does the trait blur silhouettes, colors, props, or glasses rules?
- Relationship: Does it repeat an existing pairing pattern?
- Continuity: Does it add a mystery when another mystery is still unresolved?

## Good Differentiation

A good suggestion creates contrast:

- One character notices emotional pressure while another notices technical patterns.
- One character speaks in short verdicts while another asks careful questions.
- One character uses motion, another uses stillness.
- One character has a visible prop, another has a private rule.
- One character has a flaw that causes speed, another has a flaw that causes hesitation.

## Risky Duplication

Avoid:

- Giving multiple characters the same catchphrase rhythm.
- Making every smart character a puzzle solver.
- Making every quiet character mysterious in the same way.
- Giving glasses or eyewear to non-Clever characters without owner approval.
- Adding one more running gag when a character needs emotional grounding.

## Ensemble Roles

Before approving a new trait, ask what role it supports:

- Who creates the problem?
- Who notices the problem?
- Who names the emotional truth?
- Who makes the wrong first move?
- Who changes the plan?
- Who pays the cost?
- Who gets the final button?

If the answer is already owned by another character, revise or reserve the trait.
